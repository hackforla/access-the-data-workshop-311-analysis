Data files for the analysis are generated by various notebooks.

You need to run these notebooks in this order:
  1) 311-data-cleaning.ipynb - This notebook reads data for 2021 from data.lacity.org.
  2) 311-data-cleaning-part2.ipynb -  This notebook builds on data generated by 1.
  3) build-datasets.ipynb - This notebook generates subsets for analysis.
